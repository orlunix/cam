External relay

              

                Connect Desktop to an existing CAM Hub through a relay
                endpoint. The relay forwards REST/WS to a Hub you cannot
                reach directly — you still need that Hub's API token so
                the proxied requests authenticate.

Add host manually

              

                Register a remote node by SSH. The embedded Hub stores
                host/user/port/path/env locally; the private key file
                is referenced by path only — its contents are never
                read or copied (CAM-DESK-DIRECT-018).
              

              
              
              
              
              
              

              

                
                
                
              


              

                
                
              


              

                

                  Use the running SSH agent or the default key search
